# Internal docs

This directory holds internal strategy documents that are versioned with the
repo but **not** built into the public site by Pelican.

Pelican's `pelicanconf.py` only processes `content/`, so files here are safe
from accidentally being published.

## Contents

- `marketing-audit-advertiser.md` — marketing audit findings and recommendations
  focused on the advertiser side of the marketplace.
- `marketing-audit-publisher.md` — marketing audit findings and recommendations
  focused on the publisher side.
